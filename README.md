# Expense Tracker App

A simple Expense Tracker App built with HTML, CSS, and JavaScript.

## Features

- Add Income transactions
- Add Expense transactions
- Automatically calculate total balance
- Delete transactions using the trash icon
- Real-time update of Income, Expense, and Total Balance
- Responsive and clean user interface

## Technologies Used

- HTML5
- CSS3
- JavaScript (DOM Manipulation)

## How to Use

1. Enter a description.
2. Enter an amount.
3. Select Income or Expense.
4. Click the "Add" button.
5. View the updated balance and transaction list.
6. Click the trash icon to delete a transaction.
